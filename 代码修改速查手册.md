## 一、图片存储位置说明

### 1.1 照片存储路径
- **物理路径**：`d:\AnalyticBeauty-main\public\photo\`
- **访问URL**：`http://127.0.0.1:8080/photo/文件名`
- **配置文件**：`config/upload.js`（multer上传配置）

### 1.2 数据库存储格式
- 数据库字段：`photo` 表的 `ImageURL` 字段
- 存储格式：`/photo/时间戳_随机串.jpg`（相对路径）
- 示例值：`/photo/1718456789_abc123.jpg`

## 七、修改流程总结

### 7.1 添加新字段通用流程

```
1. 数据库修改 → ALTER TABLE 添加字段
2. 后端查询 → 修改 SELECT 语句返回新字段
3. 后端新增 → 修改 INSERT 语句包含新字段
4. 后端更新 → 修改 UPDATE 语句包含新字段
5. 前端列表 → 修改表格表头和渲染逻辑
6. 前端表单 → 添加表单输入控件和验证
```

### 7.2 添加筛选条件通用流程

```
1. 前端添加筛选控件（输入框/下拉框）
2. 前端在请求参数中携带新条件
3. 后端接收新参数
4. 后端在 WHERE 子句中添加新条件
```

### 7.3 修改排序方式

```
1. 前端添加排序选择控件（如选择升序/降序）
2. 前端在请求参数中携带排序字段和方向
3. 后端接收排序参数
4. 后端修改 ORDER BY 子句
```
